---
title: A mapping table that says two inputs produce the same output is a duplication bug
date: 2026-04-28
status: active
tags: [debugging, documentation]
modules: []
---

Mapping tables that document "X produces Y" are load-bearing documentation. When a table says two distinct inputs produce the same output event, that's a duplication bug hiding in plain sight. Audit such tables before trusting them.
